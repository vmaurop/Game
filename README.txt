# Game
Hit the mosquito!


IDE:Visual studio 2015 enterprise
C#
Windows application


--external .gif(WindowsFormsApplicationGame/Recourses/mosquito_by_likara01-d5nqj2p.gif)--


A game where we want to hit the mosquito while it's moving.
Also, we have time(30 sec) and the mosquito it can be everywhere.
It is moving in a random way.

timer1: coordinates and set a random number for this (every 1000ms=1sec)

timer2:countdown(every 1000ms = 1sec)
Total duration 30sec
